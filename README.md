## 📌 1. 程序员群体的典型习惯

程序员的工作性质决定了我们容易养成以下"职业习惯"：

### ☕ 咖啡/能量饮料依赖
- 为了提神，每天摄入大量咖啡因。
- 晚上靠咖啡赶工，导致生物钟紊乱。

### 🪑 长时间久坐
- 一坐就是 6–10 小时，甚至忘记站起来。
- "这个 bug 修完再动" → 结果一动不动 3 小时。

### 🌙 熬夜 & 睡眠不规律
- "夜深人静效率高" → 凌晨 3 点还在写代码。
- 周末补觉 → 生物钟彻底紊乱。

### 🍔 不健康饮食
- 外卖三件套：炸鸡、奶茶、披萨。
- 忘记喝水，靠可乐"补充水分"。

### 🖥️ 屏幕时间过长
- 一天盯屏幕 10+ 小时，眼睛干涩疲劳。
- 蓝光暴露严重，影响褪黑素分泌。

### 😰 高压力 & 低运动
- Deadline 压力山大，却缺乏运动释放。
- "没时间健身" → 身体越来越僵硬。

> 💡 **Note**: 这些习惯看似"高效"，实则是慢性自杀。你的身体正在默默抗议！

## ⚠️ 2. 这些习惯容易导致的身体问题

### 🦴 颈椎病 & 腰椎间盘突出
- **原因**：长期低头、坐姿不良、缺乏拉伸。
- **症状**：脖子僵硬、肩膀酸痛、手臂发麻、腰痛放射至腿部。

### 👁️ 干眼症 & 视力下降
- **原因**：长时间盯屏、眨眼频率降低、蓝光伤害。
- **症状**：眼睛干涩、刺痛、视力模糊、畏光。

### 🍽️ 肥胖 & 代谢综合征
- **原因**：久坐 + 高糖高脂饮食 + 缺乏运动。
- **风险**：脂肪肝、高血脂、糖尿病前期、高血压。

### 😴 失眠 & 生物钟紊乱
- **原因**：熬夜 + 咖啡因 + 蓝光抑制褪黑素。
- **后果**：白天嗜睡、注意力下降、情绪焦虑、免疫力降低。

### ❤️ 心血管疾病风险上升
- **原因**：压力大 + 久坐 + 饮食差 → 血压、血脂、血糖异常。
- **数据**：程序员心梗年轻化趋势明显（<40 岁案例增多）。

### 🧠 焦虑 & 抑郁倾向
- **原因**：高压工作 + 社交减少 + 缺乏阳光/运动。
- **表现**：情绪低落、易怒、拖延、自我否定。

> ⚠️ **Caution**: 你以为只是"有点累"？这些症状是身体拉响的红色警报！别等住院才后悔。

## ✅ 3. 如何避免这些问题？实用解决方案

### 🪑 久坐问题 → 20-20-20 法则 + 站立办公
- **每 20 分钟**：站起来活动 20 秒（接水、拉伸、远眺）。
- **每 1 小时**：做 5 分钟全身拉伸（推荐 [颈椎操](https://youtu.be/example)、[腰椎放松](https://youtu.be/example)）。
- **升级方案**：站立办公桌（或 DIY 高桌），坐站交替（推荐 45 分钟坐 + 15 分钟站）。

> 💡 **Tip**: 用 [Stretchly](https://github.com/hovancik/stretchly) 或 [Time Out](https://www.dejal.com/timeout/) 设置定时提醒！

### 👁️ 眼睛疲劳 → 护眼模式 + 人工泪液 + 户外
- **屏幕设置**：
  - 开启 Night Light / f.lux（减少蓝光）。
  - 字体调大、对比度调高，减少眯眼。
- **护眼习惯**：
  - 使用人工泪液（无防腐剂款）缓解干涩。
  - 每天户外 30 分钟（自然光调节生物钟+护眼）。
- **设备升级**：防蓝光眼镜（争议大，但心理安慰+部分有效）。

### 🍎 饮食优化 → 健康外卖 + 定时喝水
- **替代外卖**：
  - 选择轻食沙拉、杂粮饭、清蒸菜。
  - 自备坚果、水果、酸奶当零食。
- **喝水提醒**：
  - 用 [WaterMinder](https://waterminder.app/) 或手机闹钟，每小时喝 150–200ml。
  - 戒掉含糖饮料 → 改喝柠檬水、无糖茶。

### 😴 睡眠拯救 → 固定作息 + 睡前仪式
- **黄金法则**：
  - 固定入睡/起床时间（±30 分钟），周末不赖床。
  - 睡前 1 小时禁用电子设备（或开灰度模式）。
- **助眠技巧**：
  - 热水泡脚、冥想（推荐 [潮汐 App](https://www.tide.fm/)）。
  - 卧室保持黑暗、凉爽（18–22°C 最佳）。

### 🏃‍♂️ 运动方案 → 微习惯 + 高效训练
- **从 5 分钟开始**：
  - 晨起 5 分钟拉伸，睡前 5 分钟深蹲。
  - 通勤改骑车/快走 10 分钟。
- **高效训练**（每周 3 次）：
  - HIIT（20 分钟燃脂）→ [YouTube 跟练](https://youtube.com/playlist?list=PLexample)
  - 瑜伽（改善柔韧+减压）→ [Down Dog App](https://www.downdogapp.com/)
- **目标**：每周 150 分钟中强度运动（≈ 每天 20 分钟）。

### 🧘 心理健康 → 正念 + 社交 + 休假
- **每日 5 分钟正念**：
  - 用 [Headspace](https://www.headspace.com/) 或 [小睡眠 App](https://www.xiaoshuimian.com/) 练习呼吸。
- **强制社交**：
  - 每周约朋友吃饭/运动，避免"技术宅孤独症"。
- **学会休假**：
  - 连休 3 天以上，彻底断网（真的，不回消息！）。

> ✅ **Pro Tip**: 把健康习惯写进日历！和会议一样重要。

## 🛠️ 4. 智能提醒系统

本项目包含一个智能 macOS 提醒系统，帮助你建立健康的工作习惯！

### 🎯 系统特性

- **🔄 双周轮换**: 自动切换单双周不同的日程安排
- **⏰ 智能提醒**: 支持时间段提醒（如番茄工作法）和一次性提醒
- **📊 可视化日程**: 生成美观的日程图表，直观查看安排
- **🚫 跳过功能**: 可设置特定日期跳过提醒（如周末）
- **🔧 灵活配置**: 使用 TOML 文件轻松自定义所有设置

### 📁 项目结构

```
awesome-health-habits/
├── config/                          # 配置文件目录（新结构）
│   ├── settings.toml               # 主设置文件
│   ├── settings_template.toml      # 设置模板
│   ├── odd_weeks.toml              # 单周日程
│   ├── even_weeks.toml             # 双周日程
│   └── week_schedule_template.toml # 日程模板
├── src/schedule_management/        # 源代码
│   ├── reminder_macos.py           # 主程序
│   └── __init__.py
├── tests/                          # 测试文件
├── install.sh                      # 安装脚本
└── README.md                       # 本文档
```

### ⚙️ 配置文件说明

#### 1. 主设置 (`config/settings.toml`)
```toml
[settings]
sound_file = "/System/Library/Sounds/Ping.aiff"
alarm_interval = 5  # 提醒间隔（秒）
max_alarm_duration = 300  # 最大提醒时长（秒）
skip_days = ["sunday"]  # 跳过的日期

[time_blocks]
pomodoro = 25      # 番茄工作法时长（分钟）
long_break = 40    # 长休息时长
meeting = 50       # 会议时长
exercise = 30      # 运动时长
lunch = 60         # 午餐时长
napping = 30       # 午休时长

[time_points]
go_to_bed = "上床睡觉 😴 该休息了！"
summary_time = "今天的工作结束 🎉, 总结一下"
```

#### 2. 单双周日程 (`config/odd_weeks.toml`, `config/even_weeks.toml`)
```toml
[monday]
"08:30" = "pomodoro"
"09:00" = "pomodoro" 
"09:30" = "long_break"
"13:00" = { block = "meeting", title = "团队站会" }

[common]  # 所有日期通用
"21:00" = "summary_time"
"22:45" = "go_to_bed"
```

### 🚀 快速开始

#### 1. 基础运行
```bash
# 直接运行提醒系统
uv run src/schedule_management/reminder_macos.py
```

#### 2. 生成日程可视化
```bash
# 生成单双周日程图表
uv run src/schedule_management/reminder_macos.py --view
# 查看生成的图表
open schedule_visualization/odd_week_schedule.png
open schedule_visualization/even_week_schedule.png
```

#### 3. 运行测试
```bash
# 运行所有测试
uv run pytest tests/ -v
```

#### 4. 安装为系统服务
```bash
# 使用安装脚本（推荐）
./install.sh

# 或手动安装 LaunchAgent
cp com.health.habits.reminder.plist ~/Library/LaunchAgents/
launchctl load ~/Library/LaunchAgents/com.health.habits.reminder.plist
```

### 🎨 日程类型说明

1. **时间段提醒** - 触发开始和结束提醒
   ```toml
   "09:00" = "pomodoro"  # 09:00 开始，09:25 结束
   ```

2. **一次性提醒** - 只触发一次
   ```toml
   "12:00" = "该吃午饭了！🍽️"
   "21:00" = "summary_time"
   ```

3. **自定义标题时间段**
   ```toml
   "14:00" = { block = "meeting", title = "项目评审会议" }
   ```

### ⚠️ 重要提醒

- **避免时间段重叠**：确保时间段不要重叠，否则会造成提醒冲突
- **配置文件位置**：所有配置文件现在都在 `config/` 目录下
- **权限要求**：首次运行可能需要授予系统通知权限
- **跳过日期**：在 `settings.toml` 中设置 `skip_days` 可跳过特定日期
